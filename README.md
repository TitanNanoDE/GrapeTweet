GrapeTweet
==========
A webbased Twitter client optimized for Firefox OS. 

Librarys
--------
###CryptoJS v3.1.2
GrapeTweet uses the Google hmac-sha1 from the CryptoJS Library at http://code.google.com/p/crypto-js
